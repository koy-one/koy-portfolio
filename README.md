# Koy Portfolio

Personal portfolio website for Koy, a junior frontend developer focused on responsive interfaces, clean UI structure, and polished presentation.

## Overview

This portfolio was built to present my profile, projects, skills, and contact information in a clear and modern way. The goal is to create a simple but professional frontend experience that feels strong enough for real job applications.

## Features

- Responsive layout for desktop and mobile
- Modern landing page with glass-style UI
- Project showcase section with live demo and GitHub links
- Skills and target roles sections
- Contact section with email, LinkedIn, and CV access
- Optimized project screenshots in `.webp`

## Built With

- HTML5
- Tailwind CSS
- JavaScript

## Sections

- Home
- About
- Projects
- Skills
- Contact

## Projects Included

### Netflix Clone

A frontend movie browsing experience inspired by a real streaming platform layout, focused on content presentation, reusable UI sections, and responsive structure.

### Task Manager

A task management app built with practical JavaScript features, localStorage persistence, and a simple interface designed for readability and ease of use.

## Getting Started

To run the project locally:

1. Clone the repository
2. Open the project folder
3. Launch `index.html` in your browser
